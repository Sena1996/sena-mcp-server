**Key Innovation: IO-Aware Attention**

Traditional attention requires O(NÂ²) memory for N tokens.
FlashAttention-4 uses tiling + kernel fusion to reduce this dramatically.

**Algorithm Steps:**

1. **Tiling Strategy**
   - Divide Q, K, V into blocks
   - Block size: 128 tokens
   - Fit blocks in fast SRAM
   - Minimize slow HBM access

2. **Kernel Fusion**
   - Fuse softmax + dropout + attention
   - Single GPU kernel call
   - Reduces memory round-trips
   - 92% fusion efficiency

3. **Recomputation**
   - Don't store full attention matrix
   - Recompute during backward pass
   - Trade compute for memory
   - Net speedup: 3.7x

4. **Multihead Optimization**
   - Parallel computation across heads
   - Attention heads: 32
   - Per-head efficiency: 91.2%
