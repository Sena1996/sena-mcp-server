# Show Live Progress

**Displays the SENA live progress system in your terminal**

---

To see the live progress display, run this in your terminal:

```bash
sena-progress
```

Or for interactive mode (with user options):

```bash
sena-progress --interactive
```

**Features:**
- Real-time progress bars
- Sub-process tracking
- User interaction options (1-6 keys)
- Activity log
- System status (CPU, Memory, Time)

**User Options (Interactive Mode):**
- Press 1: Skip Analysis
- Press 2: Verbose Mode
- Press 3: Quick Summary
- Press 4: Cancel
- Press 5: Pause/Resume
- Press 6: Details

The progress system will show live updates for every task SENA processes!
